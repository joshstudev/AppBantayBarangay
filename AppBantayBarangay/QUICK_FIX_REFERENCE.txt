🔧 COMMON ISSUES & INSTANT FIXES
═══════════════════════════════════════════════════════════════

ISSUE: No device/emulator shown
FIX: Tools → Android → Android Device Manager → Create New Device

ISSUE: Build fails
FIX: View → Error List (check errors)
     Right-click Solution → Restore NuGet Packages

ISSUE: "Package restore failed"
FIX: Tools → Options → NuGet → Clear All NuGet Cache(s)
     Right-click Solution → Restore NuGet Packages

ISSUE: App crashes on startup
FIX: View → Output → Select "Debug"
     Check for error messages

ISSUE: Deployment fails
FIX: Restart emulator or reconnect device
     Uninstall app from device
     Try again

ISSUE: "The application could not be started"
FIX: Clean solution
     Delete bin/obj folders
     Rebuild

═══════════════════════════════════════════════════════════════
📱 CREATE ANDROID EMULATOR
═══════════════════════════════════════════════════════════════
1. Tools → Android → Android Device Manager
2. Click "+ New Device"
3. Select "Pixel 5 - API 33" (or similar)
4. Click "Create"
5. Wait for creation
6. Select emulator from dropdown
7. Press F5
